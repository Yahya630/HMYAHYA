# converter

**HW-1: Kilometers to Miles Converter**

## الوصف
صفحة ويب بسيطة تحول المسافة من كيلومترات (km) إلى أميال (mi). تتضمن تحقق من المدخلات، زر لتحويل، وزر لمسح الحقول.

## كيفية الاستخدام
1. افتح `index.html` في متصفح حديث.
2. أدخل قيمة بالكيلومتر في الحقل.
3. اضغط `Convert to Miles` لعرض النتيجة.
4. اضغط `Clear` لمسح الحقل والنتيجة.
